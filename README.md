# Cypress Petstore Project

This project uses Cypress for end-to-end testing.

## Running Cypress

### Debug (Interactive) Mode

Run Cypress in interactive mode with:

```sh
npx cypress open
```

### Headless Mode

Run Cypress tests in headless mode with:

```sh
npx cypress run
```
